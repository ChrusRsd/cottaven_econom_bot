# ECONOV V3

Telegram bot экономики штата Монтана на `aiogram 3` и `SQLite`.

## Что умеет

- экономика только в `USD`
- переводы, чеки, топ игроков и общая статистика
- казна штата с начальными `10 000 000 USD`
- президентские налоги: до 3 налогов, суммарно не больше `700 USD`
- автоматическое списание налогов каждое воскресенье
- автоматическая выдача зарплат госслужащим два раза в день
- штрафы с оплатой через `/balance` и кнопки
- автоматический перевод просроченного штрафа в розыск через 10 дней
- розыск с пагинацией
- инвентарь, предметы и заправка транспорта
- панели владельца, администратора, президента и госструктур
- поддержка reply-команд в Telegram-группах
- уведомления в топики супергрупп и в личные сообщения госслужащим

## Быстрый старт

1. Создайте виртуальное окружение.
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Скопируйте `.env.example` в `.env`.
4. Укажите токен бота в `BOT_TOKEN`.
5. Запустите:

```bash
python main.py
```

## Важные переменные

- `OWNER_IDS` - список владельцев через запятую
- `DB_PATH` - путь к SQLite-файлу
- `TIMEZONE` - часовой пояс расписаний, по умолчанию `Europe/Moscow`
- `WEEKLY_TAX_HOUR`, `WEEKLY_TAX_MINUTE` - время воскресного удержания
- `SALARY_HOURS` - часы выплат госзарплат, по умолчанию `9,21`
- `INITIAL_USER_USD` - стартовый баланс нового пользователя, по умолчанию `200`

- `/check @user 250 Компенсация`
- `/accept 3`
- `/inventory`
- `/transfer @user Аптечка 2`
- `/refuel 25`
- `/treasuryusa`
- `/donate 500`
- `/wanteds`
- `/top`

### Президент

- `/settax 200 150 50`
- `/addgos @user`
- `/delgos @user`
- `/setgossalaries 400`
- `/withdraw @user 1000`

### Госструктуры

- `/fine @user 450 Нарушение кодекса`

## Группы и упоминания

Бот поддерживает работу в Telegram-группах. Во всех сообщениях бот использует HTML-ссылки вида `tg://user?id=...`, поэтому людей можно показывать без массовых `@`-пингов.
